# 가상클래스(Pseudo-classes)

선택자:가상클래스 {속성:속성값}으로 구성

```
selector:pseudo-class {property:value;}
```



## :link
```
a:link {}
```

## :visited
```
a:visited {}
```

## :hover
```
a:hover {}
```

## :active
```
a:active {}
```

## :not()
```
input:not() {...}
```



## :is()
```
:is(h1,h2,p) {...}
```

## :has()
```
div:has(p) {...}
```
